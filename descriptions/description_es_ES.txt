Este protector de batería para Android prolonga la duración de la batería mediante el control del estado del Wi-Fi, los datos móviles, el bluetooth y las cuentas sincronizadas. La conectividad se desactiva cuando se apaga la pantalla y se restaura a su estado original a intervalos para sincronizar mensajes, correos electrónicos, calendarios, contactos, etc. Además puede desactivarse para ahorrar aún más energía por la noche.

No estaba 100% satisfecho con las soluciones disponibles en el mercado, así que creé esta :)
Esta aplicación solo deshabilita y restaura el estado original del Wi-Fi, los datos móviles y demás. Recomendamos instalar además un widget para controlar estos. Para mí, ZDBox (https://play.google.com/store/apps/details?id=com.zdworks.android.toolbox) hizo un buen trabajo.


Esta aplicación es un software de código abierto. Deja cualquier sugerencia, informe de problemas, contribución de código y traducciones o donaciones en http://sf.net/projects/batterysaver. ¡Se agradece!
